# Usage
1. Put the given `steam_api64.dll` into the mod folder, next to `manifest.json` and `steam-pipe.dll`.
2. Create a text file named `steam_mappid.txt` next to `Sky.exe`, which contains the AppID of the game you want to mount and does not contain any other content.
3. Launch the game, whether or not using Steam.
4. Access the in-game overlay interface like a Steam game.
